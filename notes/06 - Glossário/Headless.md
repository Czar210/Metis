É a execução de um navegador de internet (como o Chromium via Playwright) de forma invisível, sem interface gráfica. Isso é fundamental para o Metis, pois permite que nossos scripts de raspagem de dados naveguem pelo Mobafire e pela Lolpedia rodando de forma silenciosa e eficiente dentro dos servidores em nuvem do GitHub Actions.
